# CS211 — Intro to Java Programming II

This repository contains personal Java projects and exercises for CS211 (Intro to Java Programming II).

Primary course text

"Building Java Programs — A Back to Basics Approach" by Stuart Reges and Marty Stepp (Pearson).

Weekly lesson summaries

- **Week 1:** Inheritance and interfaces
- **Week 2:** ArrayList and the Java Collections Framework
- **Week 3:** Recursion
- **Week 4:** Search and sorting algorithms
- **Week 5:** Stacks and queues
- **Week 6:** Implementing collection classes
- **Week 7:** Linked lists
- **Week 8:** Binary trees
- **Week 9:** Advanced data structures
- **Week 10:** Advanced data structures (continued)
